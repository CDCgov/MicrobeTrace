## Changing the Display

You can use the buttons at the top of the page to change the display. Click on a button to see the options available for the button.

The buttons are described below.
Import: Use the Import button to import data into the sequences.

Sorting: Use the Sorting button to sort the data displayed and change the order of the display. 

Filter: Use the Filter button to hide columns and sequences or reset the sequences back to the original setting after making changes.

Selection: Use the Selection button to invert columns and rows or reset the sequences back to the original setting after making changes.

Vis. Elements: Use the Vis. Elements button to hide or show different labels, sliders, meta information, or to reset the sequences back to the original setting after making changes.

Color Scheme: Use the Color Scheme button to change the color scheme of the sequences.

The current color is highlighted. 

Extras: Use the Extras button to add a consensus sequence or to jump to a column in the sequences.

Export: Use the Export button to export the sequences into other formats or share in a URL.

Help: Use the Help button to display and view information about the project, report issues, or open the user manual.
