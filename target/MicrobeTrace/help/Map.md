If your data contains geospatial columns (e.g. latitudes and longitudes), you can visualize them in the map view.

* Only rows from node csv with both lat, long columns and non-null entries. 
 
1. Click the button next to the Settings button. The system displays the WinHIVTrace menu. 
2. Click the Map option. The system displays the Map Options window.
3. Select the options to use for displaying the map. For example, you can choose the color, opacity, or choose to display tooltips in your map.
4. After selecting the map options, click OK. The system displays the data in map format.
